# For DOI Metadata Pages
Each MD under /docs/doiTemplate/DOIMarkdown will generate a page based on the DOI Metadata Template. 
The front matter of the MD file includes all the variables that are used to populate the template.

the template.html, template,css, and template.js files were used in the creation of this template, and remain for development purpposes but are not currently being edited. Any edits to the template should be made in the _layouts/doi-template.html file, or the various includes under the docs/_includes/doi-template/ folder.
