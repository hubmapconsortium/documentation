## HuBMAP Metadata Attributes by Dataset Type

A list of available dataset types (data types from multiple supported assays), with a link (**&alpha;**) to the valid metadata attributes for each dataset type. The linked assay metadata pages list all attributes, as they have occurred, across any versions of the metadata specification for the given dataset type with the most current, valid set of attributes listed first on the page. Directory schemas for each dataset type are also linked via the (**&delta;&sigmaf;**) icon in the description column.
